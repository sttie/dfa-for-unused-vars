# Dataflow анализ для C++

Тестовое задание для стажировки в JetBrains.

Анализ неиспользуемых присваиваний находится в файле AssignmentsVisitor.kt.

Удачи мне!

## Примеры работы

Пример 1 (из условия)
```
a = 1
b = a
x = 3
y = 4

while (b < 5)
  z = x
  b = b + 1
  x = 9
  y = 10
end
```

Результат:
```
Unused variables:
y = 4
y = 10
z = x
```

Пример 2 (из условия, но с всегда ложным условием цикла)
```
a = 1
b = a
x = 3
y = 4

while 10 < 5
  z = x
  b = b + 1
  x = 9
  y = 10
end
```

Результат:
```
b = a
x = 3
y = 4
```
